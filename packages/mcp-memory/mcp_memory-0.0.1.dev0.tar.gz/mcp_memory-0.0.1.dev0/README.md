## Quick Start

Add this to your `claude_desktop_config.json`

```json
{
  "mcpServers": {
    "memory": {
      "command": "uvx",
      "args": ["mcp-memory"]
    }
  }
}
```

Or use it in Python:

```shell
uv install mcp-memory
# or
pip install mcp-memory
```
